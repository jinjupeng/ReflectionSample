# C#之反射

https://www.cnblogs.com/fangyz/p/5277344.html

| 反射常用类与方法 | 说明                                                   |
| ---------------- | ------------------------------------------------------ |
| Assembly         | 通过这个类可以加载操纵一个程序集，并获取程序集内部信息 |
| EventInfo        | 这个类保存给定的事件信息                               |
| FieldInfo        | 保存给定的字段信息                                     |
| MethodInfo       | 保存给定的方法信息                                     |
| ParameterInfo    | 保存参数信息                                           |
| PropertyInfo     | 保存属性信息                                           |
| Module           | 这个类可以使你能访问多个程序集中的给定模块             |
